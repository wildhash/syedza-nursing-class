## 📚 What This Course Covers

### Part A — Global Nursing Career Pathways
- Why global demand for nurses is real (189,000+ U.S. RN openings per year)
- How Indonesian nurses can work in the **United States** — step by step
  - Documents → State Board → CGFNS/TruMerit → NCLEX-RN → English test → Employer sponsorship → VisaScreen / EB-3
- **Switzerland and Europe** — high pay, language and recognition requirements
- Where to find jobs: **Indeed, LinkedIn, hospital career pages, EURES, Craigslist**
- The **Los Angeles advantage** for Bahasa Indonesia speakers
- English and interview readiness

## 🚀 How to Use This Repo

1. Clone or download this repo
2. Open `index.html` in any browser
3. Use **← →** arrow keys or swipe to navigate slides
4. Toggle **EN / ID** in the top right corner

### Host on GitHub Pages (free)
1. Fork this repo to your GitHub account
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site is live at `https://wildhash.github.io/syedza-nursing-class/`

## 💬 Quick Prompt Library for Students

Copy and paste these into ChatGPT or Claude:

**Job search:**
> "Find me U.S. registered nurse jobs with visa sponsorship for internationally educated nurses in Florida, California, and Texas. Give me keywords, likely employers, and a checklist."

**English interview practice:**
> "Act as a hospital interviewer. Interview me for a med-surg nursing position in the U.S. Ask one question at a time and correct my English."

**Resume polish:**
> "Rewrite my nurse resume for U.S. employers. Make it clear, strong, and professional. Highlight my clinical experience and certifications."

**Visa pathway explainer:**
> "Explain the EB-3 / Schedule A visa process for a registered nurse from Indonesia who wants to work in the U.S. What are all the steps in simple language?"
